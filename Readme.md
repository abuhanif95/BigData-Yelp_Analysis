# BigData Yelp Analysis

## Creating a Branch Locally from Another Laptop

To create the `shiru` branch from another laptop, follow these steps in your terminal after pulling the repository:

1. Create and switch to the new branch:
   `git checkout -b shiru`
2. Push the branch to the remote repository:
   `git push -u origin shiru`
